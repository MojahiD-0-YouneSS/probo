shortcuts

probo.shortcuts.shortcuts

component(config)

a shortcut to Builds a Component from a Configuration Object. Handles State, Styles (CSS/BS5), and Structure in one pass.

custom(tag, content='', is_void_element=False, **attrs)

Creates a custom HTML tag string immediately.

datatable(config)

Generates a clean HTML Table.

document(config)

Assembles a full HTML Document Template. Wires up Head + Body + Layout.

head_seo(config)

Constructs a fully populated Head object with SEO metadata.

iterator(config)

shortcut Generates a list of items wrapped in a container.

layout(config)

a shortcut Generates a Reusable sementic Layout Template. Wires up Header, Footer, Sidebar, and a Main Content Wrapper. Returns a Template object ready for .swap_component().

probo_form(config)

a shortcut Builds a Form (Django or Manual).

semantic_layout(config)

a shortcut Generates a Reusable sementic Layout Template. Wires up Header, Footer, Sidebar, and a Main Content Wrapper. Returns a Template object ready for .swap_component().

set_data(*variables)

Creates a custom HTML tag string immediately.

theme(config)

Generates the :root CSS variables block. Returns the raw CSS string.

xml(config)

shortcut to Generates an XML Document or Fragment.
