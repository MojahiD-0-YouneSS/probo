request_context

probo.request.request_context

ComponentRequestContext

This class handles injecting global/context properties into Component objects that are not directly tied to a request lifecycle (e.g., static components needing context).

process_components(*extra_component_objs, **extra_props)

Applies the stored context_props (plus any extra_props) to all components.
