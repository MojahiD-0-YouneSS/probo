bs5

probo.styles.frameworks.bs5.bs5

BS5

get_cls_string(key)

Returns just the class string (Old behavior).

get_element(key, content='', **attrs)

Returns a BS5Element configured with the stored styles. Usage: bs5.get_element('btn#x2', 'Click Me')

BS5Element

A dedicated builder for Bootstrap 5 elements. Manages the 'class' attribute specifically while delegating rendering to Element.

add(*new_classes)

Fluent API to add Bootstrap classes.

include(*content, first=False, override=False)

Adds content to the element. If content is another BS5Element, it renders it.

render()

Constructs the final Element with the combined class string.
