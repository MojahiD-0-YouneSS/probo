resolver

probo.templates.resolver

TemplateResolver

collect_elements_attributes(root, load_it=False)

Traverse XML and collect tags with their merged attributes. Ignores the root tag itself.

invert_tag_attribute_dict(tag_dict, unique=True, split_classes=True)

Invert {tag: {attr: value}} into {attr: [values...]}.

Parameters:
  • tag_dict (dict) –

    dict of tags and their attributes

  • unique (bool, default: True ) –

    if True, remove duplicate values

  • split_classes (bool, default: True ) –

    if True, split 'class' attributes into separate tokens
Returns:
  • dict( dict ) –

    attribute → list of all values

merge_attributes(existing_attrs, new_attrs)

Merge new attributes into existing ones (merge class names, override duplicates).

parse_xml(xml_string)

Parse XML string and return the root element.

xml_to_tag_dict(xml_string)

Main function: XML → dict of tags → merged attributes.
