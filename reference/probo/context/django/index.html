django

probo.context.django

DjangoComponent

A declarative builder for Django Templates. Allows defining 'extends', 'blocks', raw template strings, and variables in Python.

This class is framework-agnostic at import time. Actual rendering requires Django to be installed and configured in the execution environment.

build_source()

Constructs the final raw Django Template string by assembling extensions, blocks, and content.

It combines: 1. The {% extends %} tag (if present). 2. All defined blocks. 3. Any raw template content. 4. Variable syntax translation.

Returns:
  • str( str ) –
